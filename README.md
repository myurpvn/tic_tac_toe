# Welcome to Tic-Tac-Toe Python Game

A simple python game of tic-tac-toe

## Instructions

 1. Install poetry
`pip install poetry`

 2. Install .toml file
`poetry install --no-root --without dev`

 3. Run main file
`poetry run python main.py`
